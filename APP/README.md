# 3D Geometric Modeling Application

A complete 3D geometric modeling application with a graphical interface, implementing various 3D primitives and transformations.

## Features

- Interactive 3D viewport with orthographic views (XY, XZ, YZ)
- 3D primitives: Cube, Sphere, Cylinder, Cone, Torus
- Transformations: Translation, Rotation, Scaling
- File operations: Import/Export OBJ format
- Interactive transformation controls

## Installation

1. Clone this repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```
python main.py
```

## Controls

- Left mouse button: Rotate view
- Middle mouse button: Pan view
- Right mouse button: Context menu
- Mouse wheel: Zoom in/out
